к любой команде можно дописать --help  выйдет команды для помощи 
cd test - переход в папку test (если это папка и она существует)
cd ..   - переход в родительскую папку
cd .    - переход в эту же папку (мы никуда не перейдем)
cd      -  переход в домашнюю папку (~)
pwd   - показывает аюсолютный путь до текущей папки 

mkdir test  - создание новой папки (test)
test    - это название папки 
mkdir test/new_dir  - созадиние новой папки new_d
rm -rf test         - удаление папки со всем ее содержимым 
!!!! ниикогда не использовать 
sudo rm 

nano test.txt       - открыть файл в редакторе nano 
чтобы сохранить (ctrl+s)
чтобы выйти (ctrl+x)

cat test.txt        -  просмотр содержимого файла test.txt 
ls test             -  просмотр всех файлов внутри папки 
ls -a               - просмтр скрытых файлов 
ls -l               - просмотр всех файлов с более подробной информацией 
ls -la              - просмотр всех файлов +скрытых файлов с более подробной информацией
